AnimeLad App

A modern anime discovery web app built with React. Search, explore, and keep track of your favorite anime with a personalized watchlist.

Features
	•	Browse top, action, adventure, and isekai anime categories.
	•	Search anime with live preview results.
	•	View detailed anime info: title, year, status, episodes, genres, and score.
	•	Add/remove anime from your watchlist (stored in local storage).
	•	Fully responsive design, mobile-friendly, and dark-themed.
	•	Smooth animations for hero sections, cards, and search previews.
Tech Stack
	•	Frontend: React, CSS, Vite
	•	State Management: React Context API (for watchlist)
	•	API: AniList GraphQL API
	•	Deployment: Netlify
